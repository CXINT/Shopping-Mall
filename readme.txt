项目分析
===================================
商城  电子商务
分类：B2C   C2C   B2B    O2O

商城分析
===================================
【购物流程】
首页---》浏览商品---》商品详情---》加入购物车---》结算（分支）---》订单确认页---》提交订单---》给钱---》等待发货---》确认收货---》(评价)

分支：
	有账号---》登录---》订单确认页
	没有账号---》注册---》登录---》订单确认页

准备工作
===================================
前台页面：
	首页、登录、注册、个人中心(修改个人资料、查看订单列表)、商品列表、商品详情、购物车、订单确认页(选地址)、下单成功
后台页面：
	模板框架

模块分析
===================================
前台
	用户模块
		登录、注册、个人中心(个人资料)、订单列表
	商品模块
		商品列表、商品详情
	分类模块
		遍历分类
	购物流程
		购物车 ---》 修改数量、删除商品、清空购物车
		订单确认 ---》 收货地址
		下单成功 ---》 支付按钮

后台
	用户模块
		对用户的增删改查
	分类模块
		对分类的增删改查
	商品模块
		对商品的增删改查
	订单模块
		浏览、处理订单

扩展
	评价、友情链接、网站留言。。。
